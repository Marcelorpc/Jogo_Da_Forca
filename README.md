# Jogo da Forca

Este é um jogo da Forca implementado em C#. O jogo foi projetado para ser fácil de usar e apresenta vários níveis de dificuldade.

## Características

1. Diversas categorias de palavras para tornar o jogo mais interessante e desafiador.
2. Três níveis de dificuldade para escolher: fácil, médio e difícil.
3. Sistema de contagem de acertos e erros.
4. Opção para visualizar dicas e/ou categoria (apenas no nível fácil e Medio).
5. Interface de usuário intuitiva.

## Como Jogar

Para jogar o Jogo da Forca, você precisa selecionar uma opção no menu inicial. A seguir, você precisa escolher a dificuldade do jogo. Cada nível de dificuldade tem uma quantidade específica de erros que você pode cometer antes de perder o jogo.

Durante o jogo, você tem que adivinhar as letras da palavra escolhida. Se a letra estiver na palavra, o número de acertos será aumentado. Se a letra não estiver na palavra, o número de erros será aumentado.

O jogo termina quando você adivinha toda a palavra ou quando o número máximo de erros é atingido.

## Como Instalar

Para instalar o jogo, clone este repositório e compile o código usando o ambiente .NET. O jogo pode ser iniciado executando o arquivo de saída.
